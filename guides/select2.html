How To

The select2 plugin JS is triggered by the '.js-select' class.

The data-placeholder sets the placeholder text for the select list.
N.B. There must be an empty option left first which will take on the placeholder text

<select class="js-select select2" data-placeholder="This is a select2 example" tabindex="-1">
  <option></option>
  <option>one</option>
  <option>two</option>
  <option>three</option>
</select>

The triggering JS is in main.js

if ($('.js-select').length) {
 require(['select2'], function () {
  $('.js-select').select2({
   allowClear: true
  });
 });
}
