...

To run generator-ph when starting a build:

  • Create your build folder locally e.g. buildname_html
  • Using cmd change into this directory
  • enter 'yo ph' and follow the steps as prompted
  • Important: The html files are all pulled from our BasepackageHTML git repo. When running the generator you must ensure the BasepackageHTML repo is set to public so the files can be accessed (the generator prompts you to do this before it begins). You can then set the repo back to pivate once finished.
  • Important: Once you have your local base package you will need to do a quick find and replace for: '/BasepackageHTML/'. Just delete this from all paths in your html and app.js files. This part of the path is needed for the gh-pages branch in our repo to display in the browser, can't see a way around it unfortunately.
  • Finally you're good to go. Just make sure cmd is pointing to your local build folder, run 'grunt' and start building!

Steps to update our ph generator

To work on the generator-ph you will need to clone it from our generator-ph repo to your machine locally.

While workiong locally to run the generator and check what it is doing based on your edits do the following: (source artice)

  • using cmd cd into your generator-ph local clone and enter 'npm link'
  • go to your local 'test folder' that you want to run the generator on and enter 'npm link generator-ph'
  • then run 'yo ph' on the test folder
  • Important: remember to unlink both folders after you've finished testing ('npm unlink' and 'npm unlink generator-ph' in the respective folders as outlined above).

When ready commit your edits back up to the generator-ph repo once finished (see the guide to using git)

Finally to publish these changes to the actual ph-generator you need to:

  • Change the version number in generator-ph/package.json
  • using cmd change directory to your local clone of the generator-ph and enter 'npm publish'*
  • *you do need to have publishing permissions for this command to work. Sign up for an account and then Michelle or Heather can add you - just tell them the email address you used for signing up.
  • Once published you will need to open cmd and run 'npm install -g generator-ph' - Important: make sure to notify the team as they will all need to do this step too in order to get your updates!
