"spread-values" SASS Mixin

spread values

How To

The mixin will take the css attribute and the two values you enter and then in/decrease from the first to the second value in incremenets across the breakpoints

Good for styling font-size and white space (margins and padding) across devices

h2 { @include spread-value(font-size, 20px, 30px); }

You can also specify which breakpoint to start or end the scaling at:

h2 { @include spread-value(font-size, 20px, 30px, medium, xlarge); }

The breakpoint keywords (which we've matxhed to bootstrap) are:
small: 480px,
medium: 768px,
large: 992px,
xlarge: 1200px
